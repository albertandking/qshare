# qshare
量化相关数据
类似python的akshare、tushare库

# 示例：
1. 使用sina数据源获取实时数据
> 代码:
```rust
use qshare::{sina::stock::SinaDataSource, RealTimeData};

pub mod source_data;

#[tokio::main]
async fn main() -> anyhow::Result<()>{
    // 1. 使用sina数据源获取数据
    let sina_source = SinaDataSource{};

    // 2. 指数实时数据
    let em_data = sina_source.real_time_spot_em_data().await?;

    Ok(())
}
```
